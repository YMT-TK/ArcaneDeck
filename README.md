# ArcaneDeck (奥秘卡组-开发中)

一款兼具"极简效率"与"视觉沉浸"的本地优先桌面端个人知识管理系统。

## 项目简介

ArcaneDeck 既像钢铁侠的 HUD 界面一样高效，又像魔法书一样充满仪式感，通过独特的"分类标签 + 混合排版"机制，帮助您管理碎片化知识与灵感。

## 核心价值

- **数据主权**：所有数据本地存储，用户完全掌控
- **视觉沉浸**：三种主题（科幻、魔法、极简）满足不同场景需求
- **高效管理**：混合排版适配多种内容类型
- **安全可靠**：多层防护确保数据完整性

## 技术栈

| 模块        | 技术选型                             |
| :-------- | :------------------------------- |
| **核心框架**  | Electron + React 18 + TypeScript |
| **UI 样式** | Tailwind CSS + Framer Motion     |
| **编辑器**   | Tiptap (富文本编辑)                   |
| **数据存储**  | SQLite + Prisma ORM              |
| **状态管理**  | Zustand                          |
| **图标库**   | Lucide React                     |

## 主要功能

### 🎨 主题系统

- **科幻主题**：深色磨砂玻璃背景，霓虹边框，全息扫描效果
- **魔法主题**：羊皮纸纹理，金色边框，呼吸光晕
- **极简主题**：纯白/纯黑背景，无阴影，标准字体

### 📋 内容管理

- **多种卡片类型**：便签、笔记、链接、图文
- **混合布局**：统一网格 + 自由瀑布流
- **置顶功能**：重要卡片置顶显示
- **软删除**：回收站机制，防止误删

### 💾 数据安全

- **本地优先**：所有数据存储在本地，无云端依赖
- **自定义存储路径**：支持设置数据存储位置
- **自动备份**：定期自动备份机制
- **数据库诊断**：完整性检测与修复

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建打包

```bash
# 开发环境打包
npm run build:dev

# 生产环境打包
npm run build:prod
```

## 项目结构

```
Mystery Deck/
├── arcanedeck/              # 主应用目录
│   ├── electron/            # Electron 主进程代码
│   │   ├── services/        # 业务服务层
│   │   ├── main.ts          # 主进程入口
│   │   └── preload.ts       # 预加载脚本
│   ├── prisma/              # Prisma ORM 配置
│   ├── src/                 # 渲染进程代码
│   │   ├── components/      # React 组件
│   │   ├── stores/          # Zustand 状态管理
│   │   ├── types/           # TypeScript 类型定义
│   │   └── App.tsx          # 主应用组件
│   └── package.json
```
